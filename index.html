Системы построения сетки

  1. Bootstrap / Foundation
  2. Основаны на Flexbox (Flex Grid Framework, Flexbox Grid, и т.д.)
  3. Основаны на PostCSS – LostGrid
  4. @mdo/table-grid

Хочу попробовать

  1. Chrome: chrome://flags/#enable-experimental-web-platform-features
  2. Opera: opera://flags/#enable-experimental-web-platform-features
  3. Firefox: about:config — layout.css.grid.enabled

1. Создаём grid-контейнер

			.grid-container {
				display: grid;
			}

2. Рисуем виртуальную сетку

			.grid-container {
				display: grid;
				grid-template-columns: 250px 1fr 150px;
				grid-template-rows: 100px 100px 100px;
			}

fr – fraction
